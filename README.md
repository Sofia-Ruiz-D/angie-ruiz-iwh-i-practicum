# HubSpot Custom Object Integration – Node.js

## Descripción

Aplicación desarrollada en Node.js con Express que se integra con la API REST de HubSpot para gestionar un Custom Object llamado:

Solicitudes Bot Admisiones

La aplicación permite:

- Consultar registros existentes (GET)
- Crear nuevos registros (POST)
- Visualizar datos en una interfaz simple con Pug

---

## Tecnologías utilizadas

- Node.js
- Express
- Axios
- Pug
- HubSpot CRM API (Private App Token)
